# 测试清单

## 功能链路

1. 打开页面后点击“开启摄像头”，实时预览正常显示。
2. 开启/关闭“实时增强叠层”时，预览状态切换正常。
3. 点击“拍照并增强”后，原图与增强图都可显示。
4. 点击“识别增强图文字”后，OCR 结果显示在文本框。
5. 点击“复制识别结果”后可粘贴到其他应用。

## 权限场景

1. 首次授权允许 -> 功能全通。
2. 首次授权拒绝 -> 页面提示可读，重新授权后恢复。
3. 二次进入页面 -> 权限状态识别正常。

## 手机端场景

1. iOS Safari 访问 HTTPS 地址，能调用摄像头。
2. Android Chrome 访问 HTTPS 地址，能调用摄像头。
3. 弱网下 OCR 显示加载状态，失败可重试。
